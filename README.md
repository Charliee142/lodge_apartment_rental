# lodge_apartment_rental
A secure apartment/lodge rental web application built with Django

Project Name
=============
Lodge Apartment Rental System

Description
===========
The Lodge Apartment Rental System is a full-stack web application designed to manage apartment and lodge rentals efficiently. It allows guests to browse available properties, make bookings, and manage reservations, while administrators can manage properties, bookings, and users through a secure backend.

This system is suitable for lodges, short-let apartments, guest houses, and rental agencies.

Problem it Solves
===========
Many lodge owners still rely on manual booking systems, phone calls, and paper records, which often lead to:

Double bookings

Poor customer experience

Lack of booking transparency

Security risks

No online presence

This project solves these problems by providing a secure, automated, and user-friendly rental platform.

Installation
===========
    1. Clone the Repository git clone https://github.com/Charliee142/lodge_apartment_rental.git cd lodge_apartment_rental
    2. Create Virtual Environment python -m venv venv source venv/bin/activate # Windows: venv\Scripts\activate
    3. Install Dependencies pip install -r requirements.txt
    4. Setup Environment Variables Create a .env file: SECRET_KEY=your-secret-key DEBUG=True
    5. Run Migrations python manage.py migrate
    6. Create Superuser python manage.py createsuperuser
    7. Run Server python manage.py runserver Visit:

http://127.0.0.1:8000/
